# ADR-001: Project Name

## Status

Accepted

## Context

The project required a name that reflects its current scope while allowing future growth and expansion.

The initial scope focuses on driving license services. However, the long-term vision includes the possibility of supporting additional licensing-related services such as vehicle licensing and other government licensing modules.

Therefore, the project name needed to represent the broader licensing domain rather than a single service.

## Decision

The project name will be:

**National-License-Management-System (NLMS)**

## Consequences

### Positive

* Reflects a national or government-oriented system.
* Supports future expansion beyond driving licenses.
* Represents the licensing domain as a whole.
* Remains suitable as new modules are added.

### Negative

* Longer than a domain-specific name such as Driving-License-System.
* The initial scope may appear broader than the first implementation.
